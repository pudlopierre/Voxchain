# VoxChain 🇫🇷

Plateforme citoyenne française de vote numérique sécurisé basée sur une architecture blockchain modulaire.

VoxChain vise à permettre :
- le vote national,
- les votes régionaux,
- les votes municipaux,
- les assemblées générales d'entreprises,
- et les consultations citoyennes sécurisées.

---

# Vision

Créer une infrastructure démocratique moderne :
- transparente,
- sécurisée,
- vérifiable,
- scalable,
- et accessible à tous les citoyens français.

---

# Fonctionnalités principales

## 🔐 Sécurité
- Blockchain SHA-256
- Validation OTP
- Vérification d'identité
- Traçabilité des votes
- Horodatage sécurisé

---

## 🗳️ Systèmes de vote
- Vote national
- Vote régional
- Vote municipal
- Référendums
- AG entreprises
- Sidechains spécialisées

## 📊 Transparence
